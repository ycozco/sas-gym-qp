# Obsidian

## Objetivo

Usar Obsidian para visualizar el grafo o reportes generados por Graphify.

## Estado

Herramienta opcional. No forma parte de produccion, no modifica codigo y no bloquea el desarrollo local.

## Uso

1. Ejecutar Graphify.
2. Abrir `graphify/graphify-out/` como vault en Obsidian.
3. Usar la vista de grafo para revisar relaciones entre modulos.

## Restricciones

- No guardar secretos en el vault.
- No versionar archivos temporales de Obsidian.
- No usar Obsidian como fuente unica de documentacion.

## MCP de Obsidian

Queda fuera del alcance inmediato. Solo debe evaluarse si se define servidor concreto, puerto, permisos, modo lectura/escritura, riesgos de seguridad y compatibilidad real con el flujo de Codex.
